# シフト作成アプリ（ベータ）

このリポジトリには、ブラウザで動作する職員シフト自動作成アプリが含まれています。React（CDN）と Tailwind CSS（CDN）を利用しているため、ビルド工程は不要です。`index.html` をブラウザで開くだけで利用できます。

## 利用方法

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
3. 画面上部の「自動割当」→「検証結果」を活用しながら、必要に応じてセルをクリックして調整してください。
4. 作成したシフトは CSV 出力・ブラウザ印刷・クリップボードコピーが可能です。

## 主な機能

- 早番/中遅/遅番/通常/休みの割当サポート
- 希望休・園休日・個別の must/cannot ルール
- 週上限・最小間隔・同時配置 NG（ペア）などの個別制約
- 自動割当、検証、CSV 出力、印刷、ブラウザ保存（LocalStorage）

## 開発メモ

- 依存ライブラリは CDN から読み込んでいます。
- JavaScript コードは `app.jsx` にまとまっています。
- Tailwind CSS も CDN で適用しているため、追加のビルド環境は不要です。
